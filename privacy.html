Privacy Policy

Last updated: June 2025

SyncFav is a browser extension that syncs your bookmarks to a server you own and control. This policy explains what data is involved and how it is handled.

What data the extension accesses

The extension reads your browser bookmarks in order to upload them to your server. It also stores two values in your browser's local storage: the URL of your server and your secret token. These values never leave your device except to communicate with the server URL you configured.

What data the server stores

The server stores your bookmarks as a JSON file on the server you deploy it on. The developer of SyncFav has no access to your server or your bookmarks.

Third parties

SyncFav does not send any data to third-party services. There are no analytics, no tracking, and no external APIs involved.

Your control

Because you host the server yourself, you have full control over your data. You can delete the bookmarks file on your server at any time. Uninstalling the extension removes all locally stored configuration.

Contact

For questions, open an issue on GitHub.
